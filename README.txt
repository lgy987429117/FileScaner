----------简介----------
Filesssss_Releases_1.0 版本使用的是纯Python打造的，最后使用的是Pyinstaller打包的，没有对软件包进行任何加密，你们可以自己查看源代码。所有版本都是用Bandizip压的0-仅存储。

----------教程----------
1.解压后双击打开
![屏幕捕获_2024_09_28_16_07_12_840](https://github.com/user-attachments/assets/dadd6fb9-85d7-4848-8426-1c75004c6f3c)
打开后的画面：
![屏幕捕获_2024_09_28_16_10_58_555](https://github.com/user-attachments/assets/86814669-1554-4ad2-82bf-334d402ea783)

2.键入要搜索的文件夹的绝对路径。

Tips:如果在拖动文件夹时发现绝对路径头、尾均有双（单）引号，请去掉双（单）引号以保持程序正常运行！

![录制_2024_09_28_16_17_47_455](https://github.com/user-attachments/assets/919e7cbd-f6e0-4991-8a0e-f9be5d4dd688)
然后点击回车。

3.键入要搜索的值（字符串）
![屏幕捕获_2024_09_28_16_22_02_331](https://github.com/user-attachments/assets/536d6c86-500c-4f36-9a74-157b08f2a2f6)
然后敲击回车。

4.等待搜索成功，搜索成功后按下 Enter 键。程序会将符合条件的文件全部复制至运行目录下的filesssss文件夹，并自动打开此文件夹。

End：完整操作录像（gif）
![录制_2024_09_28_16_31_54_504](https://github.com/user-attachments/assets/adfc43b0-b275-41b8-98f8-1e5d76ed5daf)
